📚 سياسة الخصوصية - مكتبة ميار

آخر تحديث: مارس 2026

1. المعلومات التي نجمعها

عند تواصلك معنا عبر Facebook Messenger، نجمع فقط:

  • معرّف المستخدم على فيسبوك (User ID)
  • محتوى الرسائل التي ترسلها لنا

2. كيف نستخدم المعلومات

نستخدم هذه المعلومات فقط للرد على استفساراتك حول منتجات مكتبتنا من أدوات مدرسية وكراسات.

3. حفظ البيانات

لا نحفظ بياناتك بشكل دائم. المحادثات تُخزَّن مؤقتاً في الذاكرة فقط أثناء جلسة المحادثة.

4. مشاركة البيانات

لا نبيع أو نشارك بياناتك مع أي طرف ثالث.

5. خدمات الطرف الثالث

نستخدم:

  • Facebook Messenger API — للتواصل مع الزبائن
  • Google Gemini API — للردود التلقائية الذكية

6. حقوقك

يمكنك في أي وقت طلب حذف بياناتك بالتواصل معنا مباشرة عبر صفحتنا.

7. التواصل

لأي استفسار حول سياسة الخصوصية، تواصل معنا عبر صفحة مكتبة ميار على فيسبوك.
